cxx:cmake

Full name:

org.codehaus.mojo:cxx-maven-plugin:0.0.6-SNAPSHOT:cmake

Description:

Goal which cmakes workspace.

Attributes:

  • Requires a Maven project to be executed.
  • Binds by default to the lifecycle phase: generate-sources.
